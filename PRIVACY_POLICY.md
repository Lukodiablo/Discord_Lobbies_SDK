# Privacy Policy

**Last Updated**: December 16, 2025

## Overview

Discord Lobbies SDK ("Extension") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your information.

## Information We Collect

### OAuth Tokens
- When you authenticate with Discord, we store your OAuth access token securely using VS Code's built-in Secret Storage API
- Tokens are **never** sent to external servers or logged
- Tokens are stored locally on your machine only

### Discord User Data
- User ID and username (displayed in UI)
- Server/Guild information you're a member of
- Lobby information you create or join
- Messages you send within lobbies

### Extension Usage
- Extension activation events (VS Code telemetry)
- Command invocations
- Error logs (for debugging only)

## How We Use Information

- **Authentication**: To authorize you with Discord's API
- **Functionality**: To display your servers, channels, and lobbies
- **Debugging**: To troubleshoot errors and improve the extension
- **No Tracking**: We do not track your activity outside the extension

## Data Storage

- All OAuth tokens are stored locally in VS Code's encrypted secret storage
- No data is transmitted to external servers
- Discord API communications use official Discord endpoints only

## Third-Party Services

- **Discord API**: The extension communicates directly with Discord's official API
- **VS Code**: Extension runs within VS Code's sandboxed environment
- **No Analytics**: We do not use any analytics or telemetry services

## Security

- OAuth tokens are managed by VS Code's secure storage
- All API communication uses HTTPS
- Sensitive data is never logged or stored in plain text

## Your Rights

- You can revoke access at any time by logging out
- Your data is not shared with any third parties
- You have full control over what data the extension accesses

## Contact

For privacy concerns regarding this extension, visit the GitHub repository or contact the developer.

## Changes to This Policy

We may update this privacy policy. Continued use of the extension after changes constitutes acceptance.
